# 법원 제출용 PDF 자동 분리 도구

> **"준비서면 몇 페이지까지야?" 한 마디면 끝.**  
> 수십 페이지짜리 PDF를 증거별로 손수 나누던 시간, 이제 그만하세요.

**변호사도, 나홀로 소송(본인 소송) 당사자도** 똑같이 쓸 수 있습니다.

---

## 이 도구가 해결하는 문제

전자소송 시스템에 서면을 등록할 때, 법원은 이렇게 요구합니다:

- 준비서면 파일 1개
- 을 제1호증 파일 1개
- 을 제2호증 파일 1개
- …

하지만 실무에서는 준비서면과 증거를 **하나의 PDF로 만들어서** Word나 한글에서 인쇄합니다.  
그걸 매번 PDF 편집 프로그램에서 열어, 페이지 범위를 확인하고, 하나씩 저장하고, 이름 붙이고…

**이 프로그램은 그 과정을 명령어 한 줄로 끝냅니다.**

---

## 나홀로 소송(본인 소송) 당사자에게

변호사 없이 직접 소송을 진행하는 분들도 전자소송 사이트에 서면을 올릴 때
변호사와 똑같은 형식 요구를 받습니다. 준비서면 따로, 증거 따로 — 파일을 하나씩 나눠
이름을 붙여 등록해야 하죠.

PDF 편집 프로그램을 따로 사지 않아도, 컴퓨터를 잘 몰라도 됩니다.

- **준비서면이 몇 페이지까지인지**만 알면 됩니다 (보통 입증방법 목록이 끝나는 페이지).
- 증거는 프로그램이 PDF 안의 글자("을 제1호증" 등)를 읽어 **자동으로 번호별로 묶어** 줍니다.
- 터미널 사용이 부담스러우면, 아래 **AI 어시스턴트(Claude Code / Codex)** 방식을 쓰면
  문장 하나 붙여넣는 것으로 끝납니다.

> 참고: 이 도구는 PDF 파일을 나눠 주는 역할만 합니다. 어떤 서류를 언제 내야 하는지,
> 증거를 어떻게 정리해야 하는지 같은 법률 판단은 도와주지 않습니다.

---

### AI에게 이렇게 말하세요 (복사해서 붙여넣기)

```
https://github.com/baryonlabs/legal-pdf-splitter 를 설치하고,
이걸 이용해서 아래 파일을 분리해줘.

파일: /Users/내이름/Documents/준비서면_2026-05-15.pdf
준비서면은 1~32페이지고, 그 다음부터 증거야.
```

---

### AI가 자동으로 해주는 것들

1. `pip install pypdf` 패키지 설치
2. GitHub에서 `split.py` 다운로드
3. PDF 분리 실행
4. 결과 파일을 원본 PDF와 같은 폴더에 저장

**파일 경로와 페이지 수 두 가지만 바꾸면, 나머지는 AI가 전부 처리합니다.**

---

## 직접 실행하는 방법 (Python이 있는 경우)

### 1단계: 필요한 패키지 설치 (최초 1회)

```bash
pip install pypdf
```

### 2단계: 실행

```bash
python split.py 준비서면파일.pdf --brief-pages 32
```

### 옵션

| 옵션 | 설명 |
|---|---|
| `--brief-pages N` | 준비서면 페이지 수 (이 페이지까지가 준비서면) |
| `--output-dir 폴더명` | 결과 파일 저장 위치 (생략 시 원본 파일과 같은 폴더) |

---

## 자동으로 처리되는 것들

| 상황 | 처리 방식 |
|---|---|
| 을 제1-1호증, 1-2호증, 1-3호증… | `을제1호증.pdf` 하나로 자동 묶음 |
| 을 제2호증-1, 2호증-2 | `을제2호증.pdf` 하나로 자동 묶음 |
| 갑호증 혼합 | 갑제X호증.pdf 별도 생성 |

증거 번호가 바뀌어도, 서브번호(1-1, 1-2…)가 붙어 있어도  
**프로그램이 자동으로 인식해서 같은 번호끼리 묶어줍니다.**

---

## 작업 흐름

```
Word/한글에서 준비서면 작성
        ↓
증거 이미지 뒤에 붙여 PDF로 저장
        ↓
이 프로그램 실행 (또는 AI에게 부탁)
        ↓
준비서면.pdf + 을제1호증.pdf + 을제2호증.pdf + … 자동 생성
        ↓
전자소송 시스템에 각 파일 등록
```

## 자주 묻는 질문

**Q. 변호사가 아니어도 쓸 수 있나요?**  
네. 나홀로 소송(본인 소송) 당사자도 그대로 쓸 수 있습니다. 전자소송에 올리는 파일 형식은
변호사든 본인이든 동일합니다.

**Q. 준비서면이 몇 페이지인지 어떻게 알아요?**  
PDF를 열어서 마지막 준비서면 페이지 번호를 확인하면 됩니다.  
보통 입증방법 목록이 끝나는 페이지가 준비서면의 마지막 페이지입니다.

**Q. 증거 번호가 매번 바뀌면요?**  
프로그램이 PDF 안의 텍스트를 읽어서 자동 인식합니다. 번호가 바뀌어도 그대로 씁니다.

**Q. 원고(갑호증)로 소송하는 경우에도 되나요?**  
네. 을호증뿐 아니라 갑호증도 자동 인식해서 `갑제X호증.pdf`로 분리합니다.

**Q. AI(Claude Code)는 어디서 구해요?**  
[claude.ai/code](https://claude.ai/code) 에서 무료로 시작할 수 있습니다.
